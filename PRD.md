# PRD: Run-Magic (러닝 코치 앱)

## 1. 제품 개요
데이터 기반의 러닝 트래킹 및 AI 코칭을 제공하여 사용자의 러닝 성과를 극대화하고 시각적 즐거움을 주는 웹 애플리케이션.

## 2. 타겟 사용자
- 자신의 러닝 데이터를 체계적으로 기록하고 싶은 러너
- 전문적인 AI 코칭을 통해 성과를 개선하고 싶은 사용자
- 멋진 그래픽과 시각화로 동기부여를 받고 싶은 사용자

## 3. 핵심 기능 (MVP)
### 3.1. 데이터 트래킹 및 대시보드
- 최근 러닝 기록 요약 (거리, 시간, 페이스)
- 성과 변화를 보여주는 다이나믹 그래프 (주간/월간)
- GPS 경로 시각화 (Mock 데이터 기반 초기 구현)

### 3.2. AI 런닝 코치
- 사용자 성향 분석 기반 오늘의 추천 훈련 (예: 인터벌, 조깅)
- 러닝 팁 및 부상 방지 정보 제공
- 실시간 성과 피드백 (텍스트/오디오 시뮬레이션)

### 3.3. 시각적 경험
- Glassmorphism 기반의 프리미엄 UI
- 성과 달성 시 축하 애니메이션 및 그래픽 효과

## 4. 기술 스택
- **Frontend**: Vite + React + TypeScript
- **Styling**: Vanilla CSS (Premium Dark Theme)
- **Visualization**: Recharts 또는 D3.js
- **State Management**: React Context API 또는 Zustand
